# GROUP BY Clause

## Overview

The `GROUP BY` clause groups rows that have the same values.

The `GROUP BY` clause helps us analyze data by grouping rows with the same values.

We can group numeric values, text values, dates, or calculated expressions.

`GROUP BY` is commonly used with aggregate functions such as COUNT(), SUM(), AVG(), MIN(), and MAX().

The `HAVING` clause filters grouped data after the GROUP BY clause.
